# Подсказка по Git

Создание репозитория:
```sh
git init
```
Получение информации от Git о текущем состоянии репозитория:
```sh
git status
```
Добавление файла к следующему коммиту:
```sh
git add
```
Создание коммита:
```sh
git commit -m "Message"
```
Вывод на экран истории всех коммитов с их хэш-кодами:
```sh
git log
```

git log --oneline
